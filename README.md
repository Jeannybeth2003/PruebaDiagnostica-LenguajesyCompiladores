Ejercicioa de lenguaje y compiladores hecho por Jeannybeth Herrera

# DESCARGAR EL JDK
Ir Al Sitio Oficial De Oracle O Tambuen Se Puede Usar Openjdk 
Descargar La Version Adecuada Para Su Sistema Operativo (Windows, Macos, Linux).

# INSTALAR EL JDK
Ejecutar El Instalador Descargado.
Seguir Las Instrucciones Predeterminadas Del Asistente De Instalación Para Asegurar La Correcta Configuración De Las Variables De Entorno.

# VERIFICAR LA INSTALACION DEL JDK
Abrir La Terminal Y Ejecutar:
Java -Version
Aqui Se Puede Apreciar La Vercion De Java Instalada

# INSTALACIÓN DEL NEATBEANS
Descargar El Jdk: Ir Al Sitio Oficial De Apache Neatbeans Y Luego Descargar La Version Adecuada
Ejecutar El Instalador Y Seguir Las Indicaciones Que Aparecen Por Pantalla
Iniciar Apache Neatbeans 

# CREAR UN PROYECTO EN APACHE NEATBEANS
Abrir Apache Neatbeans Y Seleccionar New Project
Ejegir Java Como La Principal Tecnologia 
Darle Un Nombre Al Proyecto Por Ejeplo: Ejercicio1, Ejercicio2, Ejercicio3, O Ejercicio4.

# IMPORTAR EL EJERCICIO EN APACHE NEATBEANS
Abrir El Archivo Ejercicio2 
Seleccionar Run Para Poder Correr El Codigo
El Programa Se Mpilara Y Ejecutara Los Resultados En La Consola

# DESCARGA DE EMBARCADERO DEV-C++ 5.11 Y 9.11
Ir al sitio de Embarcadero o a una fuente confiable.
Descargue la versión Dev-C++ 5.11 Y 9.11.

# INSTALACION DE EMBARCADERO DEV-C++ 5.11 Y 9.11
Ejecutar el instalador ( .exe).
Seguir las indicaciones.
Elegir Full (completa) y mantener la ubicación predeterminada.
Abrir Embarcadero Dev-C++ .

# CREAR UN PROYECTO EN Dev-C++
En Dev-C++seleccionar Archivo (File) > Proyecto (Project).
Seleccione el lenguaje: C++ .
Asignar un nombre al proyecto.
Guarde el proyecto en una carpeta específica.

# IMPORTAR EL EJERCICIO EN Dev-C++
Abrir Embarcadero Dev-C++ .
Ir a Archivo (File) > Abrir (Open) .
Navegar hasta la carpeta donde se encuentra el archivo
Seleccione el archivo de código fuente
Con el archivo abierto en el editor, compila y ejecuta con: F11(o Ejecutar > Compilar y Ejecutar ).
